# Flipkart Voice Agent

A voice-based agent for managing Flipkart orders.

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file with your configuration

## Usage

Start the agent:
```bash
npm start
```

For development with auto-reload:
```bash
npm run dev
```

## Project Structure

- `src/agent/` - Agent configuration and handlers
- `src/tools/` - Tool implementations
- `src/data/` - Mock data and database interactions
- `src/index.js` - Entry point
